# ContactProject
## Start
### Clone repository
-With HTTPS
```bash
$ git clone https://github.com/Tuhit108/ContactProject.git
```
-With SSH
```bash
$ git clone git@github.com:Tuhit108/ContactProject.git
```
### Install dependencies (node_modules)
```bash
$ yarn install
```
### Run project
```bash
$ yarn react-native start
```
```bash
$ npx react-native run-android
```
